### Colors

The official documents only mention CYMK color codes, we have extracted the RGB values from the JPG files.

#### Wero Black

| Type | Value |
| ---- | ----- |
| RGB | `29 28 28` |
| HEX/HTML | `1D1C1C` |

#### Wero Yellow

| Type | Value |
| ---- | ----- |
| RGB | `255 244 141` |
| HEX/HTML | `FFF48D` |
