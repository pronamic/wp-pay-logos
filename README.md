## Sizes

### `640` `×` `360` & `10pct` margin

```
┌──────────────────────────────┐
|         ↕ 10% = 36 ↕         |
|     ┌──────────────────┐     |
|  ↔  |      ← 512 →     |  ↔  |
| 10% | ↑              ↑ | 10% |
|  =  | 288          288 |  =  |
| 64  | ↓              ↑ | 64  |
|  ↔  |      ← 512 →     |  ↔  |
|     └──────────────────┘     |
|         ↕ 10% = 36 ↕         |
└──────────────────────────────┘
```

The `max-width` of the actual logo is `512` and the `max-height` is `288`.
